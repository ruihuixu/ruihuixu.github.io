## 核心模板（速查）

### 快慢指针模板

```python
def fast_slow_template(nums: list[int]) -> int:
    slow = 0                          # [0, slow) 始终是已处理的有效区
    for fast in range(len(nums)):
        if keep(nums[fast]):          # ← 各题唯一差异点
            nums[slow] = nums[fast]
            slow += 1
    return slow                       # 新数组长度
```

| 题号 | `keep()` 条件 | 备注 |
|------|--------------|------|
| 27   | `nums[fast] != val` | 不等于目标值则保留 |
| 26   | `nums[fast] != nums[slow - 1]`（slow 从 1 起） | 与已写入末尾不同才保留 |
| 283  | `nums[fast] != 0` | 非零保留，末尾再显式补零 |

### 对撞指针模板

```python
def collision_template(nums: list[int], target: int) -> list[int]:
    left, right = 0, len(nums) - 1
    while left < right:
        s = nums[left] + nums[right]
        if s == target:
            return [left, right]      # 找到答案
        elif s < target:
            left += 1                 # 和偏小：增大左端
        else:
            right -= 1                # 和偏大：减小右端
    return []
```

| 题号 | 核心判断 | 移动策略 |
|------|---------|---------|
| 977  | 两端平方哪个更大 | 平方较大侧向内收缩 |
| 15   | 三数之和与 0 比较（外层枚举 i） | s < 0 左移，s > 0 右移 |
| 11   | 当前面积 = min(h[l], h[r]) × (r - l) | 移动较矮的一侧 |

### 场景对比

| 维度 | 快慢指针 | 对撞指针 |
|------|---------|---------|
| 指针方向 | 同向（都从左向右） | 相向（从两端向中间） |
| 前提条件 | 无需有序 | 必须有序（或满足单调性） |
| 核心操作 | slow 作写指针，fast 作读指针 | 利用单调性确定性排除当前一侧解空间 |
| 典型场景 | 原地过滤、去重、压缩 | 两数/三数之和、容器面积 |

## 详解

### 快慢指针

#### 27. 移除元素

**核心思路**：`[0, slow)` 始终是"不含 val 的有效区"。`fast` 扫到不等于 `val` 的元素就写入 `nums[slow]` 并推进 `slow`，等于 `val` 则直接跳过。

```python
def removeElement(nums: list[int], val: int) -> int:
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow
```

图解（val=2，nums=[3,2,2,3]）：

```
fast=0: 3≠2 → nums[0]=3, slow=1
fast=1: 2=2 → 跳过
fast=2: 2=2 → 跳过
fast=3: 3≠2 → nums[1]=3, slow=2
结果：[3,3,*,*]，返回 2
```

时间 O(N)，空间 O(1)。

---

#### 26. 删除有序数组中的重复项

**核心思路**：有序数组重复元素必然相邻。用 `nums[slow-1]` 记录已写入的末尾；`fast` 遇到不同的元素才写入，从而跳过所有连续重复。

```python
def removeDuplicates(nums: list[int]) -> int:
    slow = 1   # 第一个元素必然保留，从位置 1 开始写
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow - 1]:
            nums[slow] = nums[fast]
            slow += 1
    return slow
```

图解（[1,1,2,3,3]）：

```
fast=1: 1 == nums[0]=1 → 跳过
fast=2: 2 ≠ nums[0]=1 → nums[1]=2, slow=2
fast=3: 3 ≠ nums[1]=2 → nums[2]=3, slow=3
fast=4: 3 == nums[2]=3 → 跳过
结果：[1,2,3,*,*]，返回 3
```

时间 O(N)，空间 O(1)。

#### 283. 移动零

**核心思路**：把所有非零元素前移（同 27 题逻辑），再把 `[slow, n)` 全部置零。两步分开写，逻辑更清晰且只需一次遍历加一次尾部清零。

```python
def moveZeroes(nums: list[int]) -> None:
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow] = nums[fast]
            slow += 1
    for i in range(slow, len(nums)):
        nums[i] = 0   # 有效区之后全部清零
```

时间 O(N)，空间 O(1)。

#### 844. 比较含退格的字符串

**核心思路**：从后往前双指针，用 `skip` 计数器跳过被退格的字符，逐一比对两字符串的有效字符。无需构造新字符串，实现 O(1) 空间。

```python
def backspaceCompare(s: str, t: str) -> bool:
    i, j = len(s) - 1, len(t) - 1
    skip_s = skip_t = 0
    while i >= 0 or j >= 0:
        while i >= 0:
            if s[i] == '#':
                skip_s += 1
                i -= 1
            elif skip_s > 0:
                skip_s -= 1
                i -= 1
            else:
                break
        while j >= 0:
            if t[j] == '#':
                skip_t += 1
                j -= 1
            elif skip_t > 0:
                skip_t -= 1
                j -= 1
            else:
                break
        if i >= 0 and j >= 0:
            if s[i] != t[j]:
                return False
        elif i >= 0 or j >= 0:   # 一侧已耗尽，另一侧还有有效字符
            return False
        i -= 1
        j -= 1
    return True
```

时间 O(N+M)，空间 O(1)。

#### 141. 环形链表

**核心思路**：`fast` 每次走两步，`slow` 每次走一步。若有环，fast 会在环内从后方追上 slow（龟兔赛跑）；若无环，fast 会先到 `None`。

```python
from typing import Optional

class ListNode:
    def __init__(self, val: int = 0, next: "ListNode | None" = None):
        self.val = val
        self.next = next

def hasCycle(head: Optional[ListNode]) -> bool:
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow is fast:   # 同一节点对象，确认有环
            return True
    return False
```

时间 O(N)，空间 O(1)。

### 对撞指针

#### 977. 有序数组的平方

**核心思路**：平方后最大值来自绝对值最大的两端。从结果数组末尾开始倒序填入，每次取两端平方较大的一个写入当前位置。

```python
def sortedSquares(nums: list[int]) -> list[int]:
    n = len(nums)
    res = [0] * n
    left, right = 0, n - 1
    pos = n - 1   # 从结果数组末尾开始填
    while left <= right:
        sq_l = nums[left] ** 2
        sq_r = nums[right] ** 2
        if sq_l > sq_r:
            res[pos] = sq_l
            left += 1
        else:
            res[pos] = sq_r
            right -= 1
        pos -= 1
    return res
```

图解（[-4,-1,0,3,10]）：

```
l=0(-4²=16),  r=4(10²=100) → res[4]=100, r=3
l=0(-4²=16),  r=3( 3²= 9)  → res[3]=16,  l=1
l=1(-1²= 1),  r=3( 3²= 9)  → res[2]=9,   r=2
l=1(-1²= 1),  r=2( 0²= 0)  → res[1]=1,   l=2
l=2( 0²= 0),  r=2( 0²= 0)  → res[0]=0,   r=1
结果：[0,1,9,16,100]
```

时间 O(N)，空间 O(N)。

#### 15. 三数之和

**核心思路**：排序后固定 `i`，对 `[i+1, n-1]` 做对撞双指针寻找和为 `-nums[i]` 的两数。三处去重是唯一难点：`i` 去重、`left` 去重、`right` 去重。

```python
def threeSum(nums: list[int]) -> list[list[int]]:
    nums.sort()
    res = []
    for i in range(len(nums) - 2):
        if nums[i] > 0:
            break   # 最小数已 > 0，三数之和不可能为 0
        if i > 0 and nums[i] == nums[i - 1]:
            continue   # ① i 去重：跳过与前一个相同的固定值
        left, right = i + 1, len(nums) - 1
        while left < right:
            s = nums[i] + nums[left] + nums[right]
            if s == 0:
                res.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left + 1]:
                    left += 1    # ② left 去重（先跳重复，再移动）
                while left < right and nums[right] == nums[right - 1]:
                    right -= 1   # ③ right 去重
                left += 1
                right -= 1
            elif s < 0:
                left += 1
            else:
                right -= 1
    return res
```

时间 O(N²)，空间 O(1)（排序额外 O(log N)）。

#### 11. 盛最多水的容器

**核心思路**：面积 = `min(height[left], height[right]) × (right - left)`，由短板决定。移动较高侧只会让宽度缩小且高度不增，必然不优；移动较矮侧有机会找到更高的柱子，面积才有可能变大。

```python
def maxArea(height: list[int]) -> int:
    left, right = 0, len(height) - 1
    ans = 0
    while left < right:
        h = min(height[left], height[right])
        ans = max(ans, h * (right - left))
        if height[left] < height[right]:
            left += 1    # 较矮侧向内，才有机会找到更高的柱子
        else:
            right -= 1
    return ans
```

时间 O(N)，空间 O(1)。

## 总结与扩展

双指针的本质是**用两个指针维护不变量，将 O(N²) 的嵌套枚举压缩到 O(N)**。两类模式的统一要点是：**先定义指针语义，再推导移动条件**——不变量明确后，什么时候移动哪个指针自然确定，不需要靠直觉去试。

快慢指针的不变量：`[0, slow)` 始终是合法的有效区。对撞指针的不变量：当前 `left/right` 之外的解空间已被有序性确定性排除。两类模式都不依赖额外空间，适合空间受限的场景。
