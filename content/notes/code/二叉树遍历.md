## 识别信号

> 看到什么特征，想到这篇笔记？

- 题目要求"按某顺序输出节点值"，顺序涉及前序/中序/后序 → DFS 模板，只改 `append` 的位置
- 结果类型是 `List[List[int]]`，或描述"逐层输出" → BFS 模板，`for _ in range(len(queue))` 锁层
- 题目要求"每层最后/第一个节点"（左视图、右视图） → BFS 取层尾/层首，或 DFS 先走右/左子树
- 题目要求层序结果"自底向上" → BFS 标准模板 + 结果整体翻转

## 核心模板（速查）

### 递归 DFS（三序共用框架）

```python
def dfs(node):
    if not node:
        return
    # res.append(node.val)   # ← 前序（144）：放这里，中左右
    dfs(node.left)
    # res.append(node.val)   # ← 中序（94）：放这里，左中右
    dfs(node.right)
    # res.append(node.val)   # ← 后序（145）：放这里，左右中
```

三种遍历**唯一区别**：`res.append(node.val)` 的位置，框架代码一字不差。

### 迭代 DFS（手动栈，三种写法并排）

```python
# ── 前序：先压右再压左（LIFO 保证左先弹出）──────────────────────────
stack, res = [root], []
while stack:
    node = stack.pop()
    res.append(node.val)
    if node.right: stack.append(node.right)   # 右先入，后出
    if node.left:  stack.append(node.left)    # 左后入，先出

# ── 中序：指针一路向左到底，弹出时记录，然后转右 ──────────────────────
stack, res, curr = [], [], root
while curr or stack:
    while curr:
        stack.append(curr)
        curr = curr.left
    curr = stack.pop()
    res.append(curr.val)
    curr = curr.right

# ── 后序：前序变种（先压左再压右） + 结果翻转 ──────────────────────────
stack, res = [root], []
while stack:
    node = stack.pop()
    res.append(node.val)
    if node.left:  stack.append(node.left)    # 注意顺序与前序相反
    if node.right: stack.append(node.right)
res = res[::-1]   # 中右左 → 翻转 → 左右中
```

### BFS 层序（deque 锁层）

```python
from collections import deque

queue = deque([root])
res = []
while queue:
    level_size = len(queue)        # 锁定当前层节点数，下面 for 只处理这一层
    level = []
    for _ in range(level_size):
        node = queue.popleft()
        level.append(node.val)
        if node.left:  queue.append(node.left)
        if node.right: queue.append(node.right)
    res.append(level)
```

## ⚠️ 高频错误

**1. 后序迭代压栈顺序写反，得到错误结果（最常见）**

```python
# ❌ 错误：照抄前序的压栈顺序（先压右再压左）
if node.right: stack.append(node.right)
if node.left:  stack.append(node.left)
res = res[::-1]   # 翻转后得到"右左中"，而非"左右中"

# ✅ 正确：后序变种需先压左再压右
if node.left:  stack.append(node.left)
if node.right: stack.append(node.right)
res = res[::-1]   # 翻转后才是"左右中" ✓
```

**2. BFS 忘记用 `level_size` 锁层，层级混乱**

```python
# ❌ 错误：每次只出队一个节点，无法区分层边界
while queue:
    node = queue.popleft()
    level.append(node.val)   # 把所有层混在一起

# ✅ 正确：先快照当前层大小，for 循环精确处理这一层
while queue:
    level_size = len(queue)   # 必须在 for 之前锁定
    for _ in range(level_size):
        node = queue.popleft()
```

**3. 中序迭代忘记初始化 `curr = root`，循环直接退出**

```python
# ❌ 错误：curr 初始化为 None
stack, res, curr = [], [], None
while curr or stack:   # curr=None 且 stack=[]，条件为 False，直接跳过

# ✅ 正确：curr 必须指向 root，才能走进第一次向左的 while
stack, res, curr = [], [], root
```

**4. 嵌套函数修改外部整数变量，忘写 `nonlocal`**

```python
# ❌ 错误：Python 将 count 视为局部变量，报 UnboundLocalError
count = 0
def dfs(node):
    count += 1   # 报错：赋值前引用

# ✅ 正确方案 1：nonlocal 声明
def dfs(node):
    nonlocal count
    count += 1

# ✅ 正确方案 2：用列表包装（可变对象无需 nonlocal）
count = [0]
def dfs(node):
    count[0] += 1
```

## 详解

### 144. 二叉树的前序遍历

#### 核心思路

前序 = "中左右"，先记录当前节点再递归子树。递归版三行搞定；迭代版借助栈，先压右子树再压左子树，利用 LIFO 保证左子树先弹出处理。

#### 递归版

```python
class Solution:
    def preorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []

        def dfs(node):
            if not node:
                return
            res.append(node.val)   # 中：先记录，再处理子树
            dfs(node.left)         # 左
            dfs(node.right)        # 右

        dfs(root)
        return res
```

#### 迭代版

```python
class Solution:
    def preorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        stack, res = [root], []
        while stack:
            node = stack.pop()
            res.append(node.val)
            if node.right:             # 右先入栈，后弹出
                stack.append(node.right)
            if node.left:              # 左后入栈，先弹出
                stack.append(node.left)
        return res
```

#### 图解（迭代过程）

```text
树:      1
        / \
       2   3
stack=[1]    → pop 1, res=[1],   压入 3、2
stack=[3,2]  → pop 2, res=[1,2], 无子节点
stack=[3]    → pop 3, res=[1,2,3]
```

**复杂度**：时间 O(n)，空间 O(n)（最坏链状树，栈深 n）

### 94. 二叉树的中序遍历

#### 核心思路

中序 = "左中右"，在 BST 中等价于升序输出。迭代版采用"指针一路向左撞底，弹出时记录，再转向右子树"的双循环结构，理解后不易混淆。

#### 递归版

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []

        def dfs(node):
            if not node:
                return
            dfs(node.left)         # 左：先递归到最深
            res.append(node.val)   # 中：左边走完才记录
            dfs(node.right)        # 右

        dfs(root)
        return res
```

#### 迭代版

```python
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        stack, res, curr = [], [], root   # curr 必须从 root 开始

        while curr or stack:
            while curr:               # 阶段一：一路向左，全部入栈
                stack.append(curr)
                curr = curr.left
            curr = stack.pop()        # 阶段二：左边走到头，弹出节点
            res.append(curr.val)      # 中：现在才记录
            curr = curr.right         # 阶段三：转向右子树，重复循环

        return res
```

#### 图解（迭代过程）

```text
树:    2
      / \
     1   3
curr=2 → 入栈 [2],   curr=1
curr=1 → 入栈 [2,1], curr=None（到底了）
弹出 1, res=[1], curr=None（1 无右子树）
弹出 2, res=[1,2], curr=3
curr=3 → 入栈 [3],   curr=None（到底了）
弹出 3, res=[1,2,3]
```

**复杂度**：时间 O(n)，空间 O(n)

### 145. 二叉树的后序遍历

#### 核心思路

后序 = "左右中"。迭代用**翻转法**：将前序"中左右"改成"中右左"（压栈顺序对调），最后对结果取反，即得"左右中"。无需额外复杂逻辑。

#### 递归版

```python
class Solution:
    def postorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res = []

        def dfs(node):
            if not node:
                return
            dfs(node.left)         # 左
            dfs(node.right)        # 右
            res.append(node.val)   # 中：子树全部处理完才记录

        dfs(root)
        return res
```

#### 迭代版（翻转法）

```python
class Solution:
    def postorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        stack, res = [root], []
        while stack:
            node = stack.pop()
            res.append(node.val)
            if node.left:              # 先压左（与前序相反）
                stack.append(node.left)
            if node.right:             # 再压右
                stack.append(node.right)
        return res[::-1]               # 中右左 → 翻转 → 左右中
```

**复杂度**：时间 O(n)，空间 O(n)

### 102. 二叉树的层序遍历

#### 核心思路

BFS 标准模板：`deque` + `level_size` 锁层。在每层循环开始前用 `level_size = len(queue)` 快照本层节点数，`for` 循环精确处理本层，子节点在循环内入队（留给下一层），避免层级混乱。

#### Python 代码

```python
from collections import deque

class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        queue = deque([root])
        res = []

        while queue:
            level_size = len(queue)    # 快照：锁定当前层节点数
            level = []
            for _ in range(level_size):
                node = queue.popleft()
                level.append(node.val)
                if node.left:  queue.append(node.left)
                if node.right: queue.append(node.right)
            res.append(level)

        return res
```

#### 图解

```text
树:        3
          / \
         9  20
            / \
           15   7

初始:     queue=[3]
第 1 层:  level_size=1, 出 3,     入 9、20  → level=[3]
第 2 层:  level_size=2, 出 9、20, 入 15、7 → level=[9,20]
第 3 层:  level_size=2, 出 15、7, 无子    → level=[15,7]

结果: [[3],[9,20],[15,7]]
```

**复杂度**：时间 O(n)，空间 O(n)（队列最宽为最宽一层的节点数）

### 107. 二叉树的层序遍历 II

#### 核心思路

与 102 完全相同，**唯一区别**：最终结果要从叶子层到根层排列，即对整个结果列表翻转一次：`res[::-1]`。

#### Python 代码

```python
class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        queue = deque([root])
        res = []

        while queue:
            level_size = len(queue)
            level = []
            for _ in range(level_size):
                node = queue.popleft()
                level.append(node.val)
                if node.left:  queue.append(node.left)
                if node.right: queue.append(node.right)
            res.append(level)

        return res[::-1]   # ← 与 102 唯一区别：翻转整个结果
```

### 199. 二叉树的右视图

#### 核心思路

右视图 = 每层最右边的节点。BFS 做法：在每层的 `for` 循环中，当 `i == level_size - 1` 时记录该节点。DFS 做法：先走右子树，每层第一次到达时记录（恰好是最右节点）。

#### BFS 版（推荐，直观）

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []
        queue = deque([root])
        res = []

        while queue:
            level_size = len(queue)
            for i in range(level_size):
                node = queue.popleft()
                if i == level_size - 1:        # 当前层最后一个节点
                    res.append(node.val)
                if node.left:  queue.append(node.left)
                if node.right: queue.append(node.right)

        return res
```

#### DFS 版（理解遍历本质）

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        res = []

        def dfs(node, depth):
            if not node:
                return
            if depth == len(res):          # 该深度首次访问（先走右，所以是最右节点）
                res.append(node.val)
            dfs(node.right, depth + 1)     # 先右后左：右视图优先
            dfs(node.left, depth + 1)

        dfs(root, 0)
        return res
```

**复杂度**：时间 O(n)，空间 O(n)

## 口诀

```text
DFS 三序，唯位置异：
  前序先记（中左右），中序居中（左中右），后序殿后（左右中）

迭代三招，各有妙计：
  前序：先压右，再压左（左先弹）
  中序：撞南墙，回头看右（指针向左到底再弹出）
  后序：先压左，再压右，最后翻（中右左 → 反转）

BFS 一诀：
  deque + 锁层（level_size）= 层序万金油
  取层尾（i == level_size-1）= 右视图
  翻结果（res[::-1]）= 层序 II
两族框架，只有两套代码：
  DFS 递归模板 + BFS 锁层模板，其余六题皆是变体
```

## 总结与扩展
